# Collection of Higher or Lower games

## How to compile/run

### Python
`cd Python`<br>
`python ./hl.py`

### C
[Install gnu gcc compiler](https://www.codewithharry.com/blogpost/how-to-install-gnu-gcc-compiler-on-windows/)<br>
`cd C`<br>
`gcc hl.c -o hl`<br>
`.\hl`

### Rust
`cd Rust/hl`<br>
`cargo build`<br>
`cargo run`

### C Sharp
`cd C#/hl`<br>
`dotnet build`<br>
`dotnet run`

### JavaScript
`cd JavaScript`<br>
`npm install prompt-sync`<br>
`node hl.js`

### Java
`cd Java`<br>
`javac hl.java; java hl`<br>

### Go
`cd Go`<br>
`go run hl.go`<br>
