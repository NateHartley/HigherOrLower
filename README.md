# Collection of Higher or Lower games

## How to compile/run

### Python
`cd Python`
`python ./hl.py`

### C
Must have gnu gcc compiler installed first https://www.codewithharry.com/blogpost/how-to-install-gnu-gcc-compiler-on-windows/
`cd C`
Create exe: `gcc hl.c -o hl`
Run: `.\hl`

### Rust
`cd Rust/hl`
`cargo build`
`cargo run`

### C Sharp
`cd C#/hl`
`dotnet build`
`dotnet run`
